A Python-based command-line application designed for a German coffee shop terminal. The program creates an interactive experience by prompting the user for customer details, item selection, and pricing logistics, and then instantly computes a stylized transaction receipt.
​🚀 Features
​Dynamic User Input: Uses interactive prompts to capture data on the fly rather than relying on hardcoded values.
​Type Casting: Converts raw text inputs into mathematical integers (int()) to perform calculations safely.
​Automated Logic: Automatically calculates total costs by multiplying dynamic pricing and quantity variables.
​String Manipulation: Utilizes Python f-strings and the .upper() method to format and normalize the final output into a clean, professional receipt string.

​Variables & Data Types (Strings & Integers)
​Python input() function
​Type casting with int()
​Basic arithmetic operations (*)
